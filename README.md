# Student Management System (C++ + MySQL)

## Project Description

This project is a simple console-based Student Management System built using C++ and MySQL.

## Features

* Add student record
* View student records
* Manage student data using SQL database

## Technologies Used

* C++
* MySQL

## How to Run

1. Compile the program using g++
2. Run the executable file
3. Use the menu to add or view students
